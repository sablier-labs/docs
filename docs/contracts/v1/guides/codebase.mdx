## ABIs

Depending on the web3 library you're working with, you may need to get hold of the Sablier ABIs (application binary
interfaces). The ABI acts as an interface between two program modules, one of which is the smart contract and the other
the Ethereum virtual machine code.

There are two ways to obtain it:â€Œ

1. Copy `Sablier.json` from [sablier-labs/v1-abis](https://github.com/sablier-labs/v1-abis).
2. Clone [sablier-labs/v1-protocol](https://github.com/sablier-labs/v1-protocol) and compile the contract yourself.

Here's an example for how to do step 2 with yarn and truffle:

```bash
$ git clone git@github.com/sablier-labs/v1-protocol.git
$ yarn bootstrap

The `Sablier.json` artifact should be generated in the relative `build/contracts` folder.
```
