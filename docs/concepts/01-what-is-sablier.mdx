## Protocol. Interface. Labs.

Sablier is the powerhouse of money streaming protocols. We build decentralized applications to help your money stream
from one account to another, in realtime.

- **The Sablier Protocol**: A collection of persistent, non-upgradeable smart contracts that together create a protocol
  to facilitate streaming of ERC-20 assets on Ethereum and other EVM blockchains.
- **The Sablier Interface**: A web interface that allows for easy interaction with the Sablier Protocol. The interface
  is only one of many ways to interact with the Sablier Protocol.
- **Sablier Labs**: The company that develops the Sablier Protocol, the user interfaces associated with it, and the
  documentation website you are reading right now.

### Release history

- **Sablier V1**: The first version of **The Sablier Protocol**, originally released in 2019.
- **Sablier V2.0**: The second version of **The Sablier Protocol**, released in July 2023, 4 years after the first
  version.
- **Sablier V2.1**: Released in December 2023, it was a minor upgrade over the second version.
- **Sablier V2.2**: The latest version of **The Sablier Protocol**.
