A generic entity for tracking protocol actions. There may be multiple actions for a single tx.

```graphql
type Action {
  id: String!
  subgraphId: BigInt!
  block: BigInt!
  chainId: BigInt!
  from: Bytes!
  hash: Bytes!
  timestamp: BigInt!
  category: ActionCategory!
  contract: Bytes!
  fee: BigInt
  stream: Stream
  addressA: Bytes
  addressB: Bytes
  amountA: BigInt
  amountB: BigInt
}
```
