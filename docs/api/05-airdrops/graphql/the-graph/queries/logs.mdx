Query execution logs emitted by the subgraph during indexing. Results are sorted by timestamp in descending order
(newest first).

```graphql
_logs(
  level: LogLevel
  from: String
  to: String
  search: String
  first: Int = 100
  skip: Int = 0
  orderDirection: OrderDirection = desc
): [_Log_!]!
```

### Arguments

#### [<code style={{ fontWeight: 'normal' }}>\_logs.<b>level</b></code>](#level)<Bullet />[<code style={{ fontWeight: 'normal' }}><b>LogLevel</b></code>](/docs/api/05-airdrops/graphql/the-graph/enums/log-level.mdx) <Badge class="badge badge--secondary " text="enum"/> \{#level\}

Filter logs by severity level. Only logs at this level will be returned.

#### [<code style={{ fontWeight: 'normal' }}>\_logs.<b>from</b></code>](#from)<Bullet />[<code style={{ fontWeight: 'normal' }}><b>String</b></code>](/docs/api/05-airdrops/graphql/the-graph/scalars/string.mdx) <Badge class="badge badge--secondary " text="scalar"/> \{#from\}

Filter logs from this timestamp onwards (inclusive). Must be in RFC3339 format (e.g., '2024-01-15T10:30:00Z').

#### [<code style={{ fontWeight: 'normal' }}>\_logs.<b>to</b></code>](#to)<Bullet />[<code style={{ fontWeight: 'normal' }}><b>String</b></code>](/docs/api/05-airdrops/graphql/the-graph/scalars/string.mdx) <Badge class="badge badge--secondary " text="scalar"/> \{#to\}

Filter logs until this timestamp (inclusive). Must be in RFC3339 format (e.g., '2024-01-15T23:59:59Z').

#### [<code style={{ fontWeight: 'normal' }}>\_logs.<b>search</b></code>](#search)<Bullet />[<code style={{ fontWeight: 'normal' }}><b>String</b></code>](/docs/api/05-airdrops/graphql/the-graph/scalars/string.mdx) <Badge class="badge badge--secondary " text="scalar"/> \{#search\}

Search for logs containing this text in the message. Case-insensitive substring match. Maximum length: 1000 characters.

#### [<code style={{ fontWeight: 'normal' }}>\_logs.<b>first</b></code>](#first)<Bullet />[<code style={{ fontWeight: 'normal' }}><b>Int</b></code>](/docs/api/05-airdrops/graphql/the-graph/scalars/int.mdx) <Badge class="badge badge--secondary " text="scalar"/> \{#first\}

Maximum number of logs to return. Default: 100, Maximum: 1000.

#### [<code style={{ fontWeight: 'normal' }}>\_logs.<b>skip</b></code>](#skip)<Bullet />[<code style={{ fontWeight: 'normal' }}><b>Int</b></code>](/docs/api/05-airdrops/graphql/the-graph/scalars/int.mdx) <Badge class="badge badge--secondary " text="scalar"/> \{#skip\}

Number of logs to skip (for pagination). Default: 0, Maximum: 10000.

#### [<code style={{ fontWeight: 'normal' }}>\_logs.<b>orderDirection</b></code>](#order-direction)<Bullet />[<code style={{ fontWeight: 'normal' }}><b>OrderDirection</b></code>](/docs/api/05-airdrops/graphql/the-graph/enums/order-direction.mdx) <Badge class="badge badge--secondary " text="enum"/> \{#order-direction\}

Sort direction for results. Default: desc (newest first).

### Type

#### [<code style={{ fontWeight: 'normal' }}><b>\_Log\_</b></code>](/docs/api/05-airdrops/graphql/the-graph/objects/log.mdx) <Badge class="badge badge--secondary " text="object"/>

A log entry emitted by a subgraph during indexing. Logs can be generated by the subgraph's AssemblyScript code using the
`log.*` functions.
