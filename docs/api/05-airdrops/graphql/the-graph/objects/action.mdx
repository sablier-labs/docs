A generic entity for tracking protocol actions. There may be multiple actions for a single tx.

```graphql
type Action {
  id: String!
  subgraphId: BigInt!
  block: BigInt!
  chainId: BigInt!
  from: Bytes!
  hash: Bytes!
  timestamp: BigInt!
  campaign: Campaign!
  category: ActionCategory!
  fee: BigInt
  claimAmount: BigInt
  claimIndex: BigInt
  claimRecipient: Bytes
  claimTo: Bytes
  claimStreamId: String
  claimTokenId: BigInt
  clawbackAmount: BigInt
  clawbackFrom: Bytes
  clawbackTo: Bytes
  vcaForgoneAmount: BigInt
}
```
