A log entry emitted by a subgraph during indexing. Logs can be generated by the subgraph's AssemblyScript code using the
`log.*` functions.

```graphql
type _Log_ {
  id: String!
  subgraphId: String!
  timestamp: String!
  level: LogLevel!
  text: String!
  arguments: [_LogArgument_!]!
  meta: _LogMeta_!
}
```
