The severity level of a log entry. Log levels are ordered from most to least severe: CRITICAL &#x003E; ERROR &#x003E;
WARNING &#x003E; INFO &#x003E; DEBUG

```graphql
enum LogLevel {
  CRITICAL
  ERROR
  WARNING
  INFO
  DEBUG
}
```

### Values

#### [<code style={{ fontWeight: 'normal' }}>LogLevel.<b>CRITICAL</b></code>](#critical) \{#critical\}

Critical errors that require immediate attention

#### [<code style={{ fontWeight: 'normal' }}>LogLevel.<b>ERROR</b></code>](#error) \{#error\}

Error conditions that indicate a failure

#### [<code style={{ fontWeight: 'normal' }}>LogLevel.<b>WARNING</b></code>](#warning) \{#warning\}

Warning conditions that may require attention

#### [<code style={{ fontWeight: 'normal' }}>LogLevel.<b>INFO</b></code>](#info) \{#info\}

Informational messages about normal operations

#### [<code style={{ fontWeight: 'normal' }}>LogLevel.<b>DEBUG</b></code>](#debug) \{#debug\}

Detailed diagnostic information for debugging
