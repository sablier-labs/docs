Creating streams in bulk is possible using the SablierBatchLockup contract. See
https://github.com/sablier-labs/lockup/blob/v2.0/src/SablierBatchLockup.sol

Note: the entity can be immutable because a batch is only updated in the same block. See
https://thegraph.com/docs/en/subgraphs/developing/creating/ql-schema/#defining-entities

```graphql
type Batch {
  id: String!
  hash: Bytes
  timestamp: BigInt
  batcher: Batcher
  position: BigInt
  size: BigInt!
  streams(
    skip: Int = 0
    first: Int = 100
    orderBy: Stream_orderBy
    orderDirection: OrderDirection
    where: Stream_filter
  ): [Stream!]!
}
```
