Creating streams in bulk is possible using the Sablier batch contracts.

```graphql
type FlowBatch {
  id: String!
  hash: Bytes
  timestamp: BigInt
  batcher: FlowBatcher
  position: BigInt
  size: BigInt!
  streams(
    skip: Int = 0
    first: Int = 100
    orderBy: FlowStream_orderBy
    orderDirection: OrderDirection
    where: FlowStream_filter
  ): [FlowStream!]!
}
```
