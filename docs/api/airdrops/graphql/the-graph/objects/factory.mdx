Entity for Merkle factories, which deploy campaigns.

```graphql
type Factory {
  id: String!
  address: Bytes!
  alias: String!
  campaignCounter: BigInt!
  chainId: BigInt!
  campaigns(
    skip: Int = 0
    first: Int = 100
    orderBy: Campaign_orderBy
    orderDirection: OrderDirection
    where: Campaign_filter
  ): [Campaign!]!
}
```
