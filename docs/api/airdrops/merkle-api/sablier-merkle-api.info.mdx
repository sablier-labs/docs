REST API for generating and verifying Merkle trees used in Sablier airdrops.

This service handles tree creation, storage, and data retrieval for both the Sablier Interface and third-party
applications.

## Architecture

The backend is written in Rust and can run locally or in self-hosted environments using warp, as well as on Vercel with
its rust runtime.

You can integrate the backend functionality using our official endpoints, but we recommend self-hosting for improved
reliability.
